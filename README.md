### âš™ï¸ Setup

1. **Clone the repository**  
   ```bash
   git clone https://github.com/kshitij7704/E_Commerce_Recommendation_System
   cd echomart
   ```

2. **Set up the backend**
   ```bash
   pip install -r requirements.txt
   python app.py
   ```

   ```bash
   cd Angular-Frontend
   npm install
   ng serve
   ```
